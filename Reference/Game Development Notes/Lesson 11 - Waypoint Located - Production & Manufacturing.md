# Lesson 11: Production & Manufacturing

## TL;DR
- Two manufacturing paths: 1,000+ copies = factory (China/Europe). <500 copies = print-on-demand (Game Crafter).
- Major manufacturers: LudoFact (Germany), Panda (China, ~90% of Kickstarter games, most expensive), Longshore, LongPack.
- Total timeline: Quoting (6+ months ahead) → Material samples (3+ months) → Files & proofing (2-3 weeks) → Pre-production (2-4 weeks) → Printing/assembly (10-16 weeks) → Shipping (2-3 months) = 5-9 months file-to-delivery.
- Get quotes from 2-3+ factories. Always request material samples (white sample + digital proofs) and physical proofs.
- Boxes are hardest to fix—be meticulous with box files. Figures are most expensive and longest to produce.
- Component cost hierarchy: tokens < punch board < wood ≈ plastic < acrylic < figures.
- Custom inserts cut during budgeting. Stock components cheap; custom components require tooling (expensive, distinctive).
- Print-on-demand: higher per-unit cost, less quality control, flat fee model, limited complexity.
- Reprints skip proofing/pre-production but still 4-6 months. Payment typically at file submission.

## Core Concepts
- **Manufacturing Paths** — Large runs (1,000+) = factory with economies of scale. Small runs (<500) = PoD, higher per-unit cost, simpler process, faster turnaround. Factories = low-cost per unit, slow, high complexity possible. PoD = high per-unit cost, fast, limited options.
- **Timeline Reality** — File submission to games in hand: 5-9 months. Most delays happen during this phase. Shipping alone = 2-3 months. Quoting must start 6+ months before desired print date.
- **Quality Control Gates** — White sample (unprinted, feel quality). Digital proofs (PDFs verify files). Physical proofs (printed samples check color/trim). Pre-production sample (proper materials, correct cuts). Skipping gates risks surprises.
- **Component Cost Drivers** — Stock components (standard dice) = cheap. Custom components = expensive because tooling required (metal molds for figures, dies for punch patterns). Figures = most expensive + longest to produce (sculpting + metal tool).
- **Material Selection** — Premium vs. affordable decision made early, affects all choices. $100 game = deluxe components. <$10 game = strip down to essentials. Think through price point before designing component list.
- **Overruns** — Factories overproduce 2-5% for replacements. Expect ~1% minor errors. Major mistakes (10%+ assembled wrong) = factory reproduces and ships replacements at no cost (honorable factories).

## Design Principles
- If large print run (1,000+) → quote 6+ months ahead, allow 5-9 months for delivery
- If small print run (<500) → use PoD, accepts lower per-unit margins, useful for testing
- If custom components → start quoting/sampling early (figures can take months)
- If first game → always request physical proofs; digital ≠ physical quality
- If budget tight → custom inserts are first cut; gameplay > presentation
- If premium game → decide early; affects every component choice (cost, quality, sourcing)
- Prioritise box quality → boxes are hardest to fix if error occurs; be meticulous with files
- If reordering → skip proofing/pre-prod, still 4-6 months total

## Heuristics (Fast Decision Rules)
- **Quote Timeline**: 6+ months before print = minimum lead time for quotes to manufacturing.
- **Samples Rule**: Always get white sample + digital proofs + physical proofs (especially first game). Cost is negligible compared to fixing problems after printing.
- **Component Fast Track**: Figures need earliest start (sculpting + metal tool). Standard stock components = shortest timeline.
- **Box Vigilance**: Triple-check box files. Fixing wrong barcode = stickering every copy in warehouse. Wrong box dimensions = entire run potentially unusable.
- **Reprint Speed**: Skip proofing/pre-prod = 4-6 months. Full process = 5-9 months.
- **Component Hierarchy by Cost**: Tokens < punch board standees < wood ≈ plastic < acrylic standees < plastic figures (most expensive).

## Design Frameworks
- **Manufacturing Timeline (5-9 months file→delivery)**:
  1. **Quoting (6+ months ahead)**: Contact 2-3+ factories, provide component list and timeline. Compare prices, request material samples.
  2. **Material Samples (3+ months ahead)**: Receive white sample (unprinted) and samples of card stock, dice, other materials. Evaluate quality, adjust material choices if needed. Back-and-forth with factory if custom components needed.
  3. **File Submission & Proofing (2-3 weeks)**: Lock down final files, submit. Factory reviews, sends digital proofs (PDFs). You review for accuracy. Request physical proofs (printed samples). Check colors, trim, overall quality.
  4. **Pre-Production (2-4 weeks)**: Factory makes printing plates and specialized dies (e.g., punch board dies). Setup work before mass production.
  5. **Printing & Assembly (10-16 weeks)**: Mass production—print components, cut, shrink wrap, assemble into boxes. Complexity determines timeline.
  6. **Shipping (2-3 months)**: Factory → boat/train → dock → warehouse. Multiple handoffs. East-West shipments (China-US) = 6-12 weeks.
  7. **Warehouse to Customers (3-4 weeks additional)**: If self-pub, you handle fulfillment. If using distributor/publisher, they handle.

- **Quality Control Checkpoints**:
  - **White Sample**: Unprinted game. Feel card thickness, box weight, overall construction. Ask: Am I happy with this production quality?
  - **Digital Proofs**: PDFs showing all components and press layouts. Check fonts, colors haven't corrupted, cards trimmed correctly.
  - **Physical Proofs**: Actual printed sample. Colors can vary dramatically from digital. Check for wash-out, darkness, trim accuracy.
  - **Pre-Production Sample**: Proper materials (not uncut sheets), punch board cut correctly, everything in final form.

- **Component Cost & Timeline Matrix**:
  - **Stock components** (standard dice, tokens): Cheap, immediate availability, short timeline.
  - **Custom components** (screened dice, meeples): Expensive, require tooling, longer timeline (weeks to months).
  - **Wood pieces**: Price ≈ plastic, require molds, simpler shapes only (no complex geometry).
  - **Acrylic standees** (printed clear plastic): More expensive, average timeline.
  - **Plastic figures**: Most expensive, longest timeline (months). Requires sculptor + metal tool. Distinctive but costly.
  - **Punch board standees**: Cheapest standee option, fastest production, includes plastic base.
  - **Custom inserts**: Usually cut during budgeting. Nice-to-have, not necessary for gameplay. ~60% of designers say "nice but not necessary."

- **Self-Publishing Decision (Timeline + Complexity)**:
  - **Goal = run game company**: Self-publish, learn full process, expand with each game.
  - **Goal = make one game, move on**: Pitch to publisher (they handle manufacturing).
  - **Goal = test market first**: Use PoD, gather sales data, approach distributors with proof of concept.

## Common Pitfalls
- **Late quoting** → Starting manufacturing conversation <6 months before desired print date. Fails because: insufficient lead time for samples, design decisions, file prep. Fix: Quote 6+ months ahead, treat timeline as hard deadline.
- **Skipping physical proofs** → Relying on digital proofs only. Fails because: color rendering digital ≠ physical printing; trim accuracy differs. Fix: Always request physical proofs, especially first game.
- **Over-custom components** → Designing with many unique pieces when stock alternatives exist. Fails because: cost balloons, timeline extends. Fix: Use stock components, add one custom signature piece if budget allows.
- **Inadequate box file review** → Not triple-checking box files. Fails because: boxes are hardest/most expensive to fix post-printing. Fix: Review box files obsessively. Get factory feedback on spacing, trim.
- **Underestimating timeline** → Expecting fast turnaround. Fails because: 5-9 months file-to-delivery is baseline. Add time for factory delays, shipping delays. Fix: Plan backwards from launch date, allow buffer.
- **Ignoring overrun handling** → Not discussing what happens with 2-5% overproduction. Fails because: unclear who owns extras, no replacement parts set aside. Fix: Clarify during quoting: are overruns included? How many replacement sets reserved?
- **Multiple regional shipments without planning** → Shipping to US + Canada + EU warehouses simultaneously. Fails because: inventory guessing wrong, stuck with unsold stock. Fix: Start in home region, expand based on distributor interest.

## Application
- **Starting manufacturing**: Get quotes from 2-3+ factories 6+ months before desired print date. Request material samples (white + digital proofs). Budget 2-4 weeks for sample evaluation.
- **Quality control discipline**: Request physical proofs no matter cost. One surprise at delivery (wrong colors, misaligned trim) could ruin launch. Physical proofs catch these.
- **Component strategy**: Decide premium vs. affordable early. Standard stock = acceptable cost/timeline. One custom signature piece (screened dice, meeple variant) = distinctive without huge cost.
- **Box meticulous review**: Treat box files as critical path. Get factory feedback on trim, spacing, barcode placement. Triple-check before submission.
- **Reprintability**: Design components to allow reprints without major retooling. Consistent punch board patterns across sheets = one die, not many.
- **Timeline buffer**: Plan backwards from launch. File submission = 5-9 months before customer delivery. Add 1-2 weeks buffer for unforeseen delays.
- **Reprint decisions**: Skip proofing/pre-prod on reprints, but still 4-6 months. Order reprints when inventory = 3-4 months supply remaining.
