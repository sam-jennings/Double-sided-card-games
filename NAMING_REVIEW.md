# Game Name Review

*June 2026 — a review of all game names in the collection for clarity. One deck runs every game, so the **names** are the only thing telling players which game is which. They have to be distinct from each other and suggest what the game is. Two problems below: names that collide with each other, and names that don't connect to their rules.*

> **RESOLVED — applied June 2026.** Every recommended rename below was carried out across the repository: **FALSE FACE → FORKED TONGUE**, **TWO-FACED → SLEEPER**, **TURNOVER → WILDFIRE** (its chain-3 variant renamed **INFERNO** to free the word), **CABAL → SNARE**, **MERIDIAN → RELAY**. FACE VALUE, TURNCOAT, TRIGON, and GLEAN were kept. The analysis below is preserved as the original review and rationale, so **game names in it refer to the pre-rename state.** The convention now lives in [`.kiro/steering/naming.md`](.kiro/steering/naming.md); the canonical index is [`COLLECTION_OVERVIEW.md`](COLLECTION_OVERVIEW.md).

## Problem 1 — Names that collide with each other

### A. The "FACE" pile-up — the most serious issue

Three games share the word, **and the deck itself is "double-faced."** Worse, the two closest names are *also the same genre*:

| Name | Genre | The clash |
|---|---|---|
| **FALSE FACE** | bluffing (3–6P) | same word **and** same genre as Face Value |
| **FACE VALUE** | bluffing (2P) | same word **and** same genre as False Face |
| **TWO-FACED** | social deduction (4–6P) | third "face"; also betrayal-themed like Turncoat |

"Was the 2-player bluff *False Face* or *Face Value*?" is a question players *will* ask. "Face" has stopped distinguishing anything.

**Recommendation — keep one, rename two.**

- **Keep FACE VALUE.** It's the cleverest of the three — "take it at face value" *is* the mechanic (the shown face is your only claim). It earns the word.
- **FALSE FACE → FORKED TONGUE.** The game is lying aloud and getting caught when the ledger catches up; "forked tongue" says *liar* instantly and drops "face" entirely. *Alternates:* TALL TALES (lighter, more obviously party), PROVENANCE (nods to the verifiable-claim record, but quieter).
- **TWO-FACED → SLEEPER.** A hidden allegiance that only surfaces at the reckoning — exactly a sleeper agent. Sits cleanly beside TURNCOAT without overlapping (a sleeper hides; a turncoat switches in the open). *Alternates:* THE PRETENDER (false-loyalty / ruling-sign throne motif), FIFTH COLUMN (secret-faction angle, slightly obscure).

### B. The "TURN" pair

**TURNCOAT** and **TURNOVER** both start "TURN-", are similar length, and are both "about flipping." (TWO-FACED's core action is even *"Turn a coat."*)

**Recommendation — keep TURNCOAT, rename TURNOVER → WILDFIRE.** Turncoat is a near-perfect name for the defection game. "Turnover" is the weaker word anyway (it also means a pastry, staff churn, a sports turnover). **WILDFIRE** is already the name of the game's best variant, and a spreading fire maps onto the match-and-chain cascade that forces the next player — fast, loud, ten minutes, a name that moves. *Alternates:* TINDERBOX, BRUSHFIRE.

### C. Softer notes (optional)

- **T-heavy shelf.** Eight names start with T (Twelve Trials, Trigon, Turncoat, Turnover, Two-Faced, The Orrery, The Council, The Unplayed Pair). Renaming Turnover + Two-Faced above already thins this.
- **"THE ___" inconsistency.** Only three names take an article (The Orrery, The Council, The Unplayed Pair). Dropping them (ORRERY, COUNCIL) would make the set consistent — purely cosmetic.

## Problem 2 — Names that don't connect to the rules

### Worst fit (both are still working titles — low cost to fix)

- **CABAL → SNARE.** "Cabal" sounds like a hidden-role *social* game (it reads as a sibling of Two-Faced); the actual game is a silent 2-player geometric trap — don't be the one to complete the forbidden shape, only steer it onto your opponent. Existing alts (Schism, Quorum) all sound political too. **SNARE** carries that "don't get caught" tension in one word. *Alternates:* GORDIAN (leans into the inescapability, more mythic), IMPASSE. *(Avoid TRIAD — too close to Trigon.)*
- **MERIDIAN → RELAY.** A meridian is a line of longitude; it evokes nothing about roads, racing, or a network. The game is a route-race where connection is handed along a growing web — **RELAY** gives it the race energy the current name lacks. *Alternates:* CARAVAN (warmer, trade-routes-across-a-map flavour), JUNCTION (plainer, more network-y).

### Obscure, but the meaning *is* right (keep, or simplify only if accessibility matters)

These aren't wrong — they're just words many players won't know. The fit is exact once explained.

| Name | Means | Fit | If you want plainer |
|---|---|---|---|
| **OUROBOROS** | serpent eating its tail | the game *is* a self-closing serpent | THE LONG SERPENT, SERPENTINE |
| **THE ORRERY** | clockwork model of orbits | you sort cards into "orbits" | ORBITS, CLOCKWORK *(a playtester misread "orbit" — worth watching)* |
| **JANUS** | two-faced Roman god | each card shows two faces at once | — *(strong, distinctive; keep)* |
| **TRIGON** | a triangle / a three | three-in-a-row capture | — *(brandable; just note it hides "grid combat")* |

My take: **keep Ouroboros, Janus, and Trigon** — they're distinctive and the fit is real. **The Orrery** is the one to watch, because a player already misread it at the table.

## Recommended renames

| Current | → Recommended | Why | Alternates |
|---|---|---|---|
| **FALSE FACE** | **FORKED TONGUE** | says *liar* instantly; drops the "face" collision | Tall Tales · Provenance |
| **TWO-FACED** | **SLEEPER** | hidden allegiance revealed at the reckoning; distinct from Turncoat | The Pretender · Fifth Column |
| **TURNOVER** | **WILDFIRE** | already its best variant; fire = the match-and-chain cascade | Tinderbox · Brushfire |
| **CABAL** *(concept)* | **SNARE** | a silent 2P trap, not a social game; "don't get caught" | Gordian · Impasse |
| **MERIDIAN** *(concept)* | **RELAY** | route-race energy a "meridian" has none of | Caravan · Junction |
| THE ORRERY *(watch)* | *Orbits (only if misread again)* | a playtester already misread "orbit" | Clockwork |

*Note: SLEEPER and SNARE both start with S, but they're a 6-player social-deduction game and a 2-player abstract — no real confusion. Swap SNARE → GORDIAN if you'd rather avoid even that.*

## Priority

1. **Break the FACE collision** — False Face → Forked Tongue, Two-Faced → Sleeper; keep Face Value. *(Highest impact — three games, two of them the same genre.)*
2. **Break the TURN collision** — Turnover → Wildfire; keep Turncoat.
3. **Fix the two misleading concept titles** — Cabal → Snare, Meridian → Relay. *(Cheap: neither has a rulebook yet.)*
4. **Optional** — watch The Orrery; drop "THE" articles for consistency if wanted.

Eight of the developed games need no change. The shortlist that actually wants a decision: **False Face, Two-Faced, Turnover** (built) and **Cabal, Meridian** (concepts).
